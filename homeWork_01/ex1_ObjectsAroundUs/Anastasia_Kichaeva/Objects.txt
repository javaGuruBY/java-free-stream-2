    1. Объект - носок.
Состояние :
  - надет/снят
  - в стирке
  - мокрый/сухой
  - вонючий/ароматный
  - целый/рваный
Поведение: 
  - согревает(в надетом состоянии позволяет согреть ногу)
  - растягивается(с каждой стиркой становится больше)
  - изнашевается(при длительном ношении образуются дырки, теряется вид)
      2. Объект - воздушный шарик.
Состояние :
  - цвет
  - форма
  - размер
  - наполненный/пустой
Поведение:
  - лежит на земле(при наполнении воздухом)
  - лежит на земле(в ненаполненном состоянии)
  - взлетает вверх(при наполнении гелием)
    3. Объект - горшок.
Состояние :
  - цвет
  - размер
  - форма
  - наполненный/пустой
Поведение : 
  - воняет(при наполнении содержимим ребенка)
  - стоит(ждет своего звездного часа, пока ребенок дорастет)
    4. Объект - дерево.
Состояние :
  - размер
  - вид(цвет, форма, размер листа, высота ствола и т.п.)
  - возраст
Поведение:
  - растет(при наличии благоприятных условий)
  - гибнет(при отсутсутствии благоприятных условий)
  - фотосинтез(см. Вики)
    5. Объект - гвоздь.
состояние :
  - цвет
  - размер
  - забит/не забит
Поведение :
  - скрепляет(в забитом состоянии)
  - гнется/ломается(в случае неудачного забивания или кривых рук)
  - лежит в коробочке с инструментами(если над ним не совершается никаких дествий)
    6. Объект - туалетная бумага.
Состояние :
  - цвет
  - запах
  - состав(100% целлюлоза/ вторичное сырье)
  - слоистость
  - текстура
  - дизайн
Поведение :
  - заканчивается (в самый неподходящий момент)
    7. Объект - чебурек.
Состояние :
  - температура
  - форма
  - размер
  - текстура
  - вкус
  - цвет
  - срок годности
  - качество
Поведение :
  - издает запах(ароматный/горелый/испорченный)
  - съеден
  - портится(покрыт плесенью, имеет неприятный запах, засох)
  - выкинут
  -разлагается
    8. Объект - младенец.
Состояние :
  - вес
  - рост
  - пол
  - цвет
  - возраст
  - настроение
  - закрытие потребностей(сытый/голодный, чистый/грязный, здоровый/больной, выспавшийся/уставший, наигранный/испытывающий дефицит внимания)
Поведение :
  - веселится/играет/радуется(при закрытии потребностей)
  - грустит/плачет/кричит(в состоянии незакрытых потребностей)
  - растет
  - ест
  - справляет нужду
  - играет
  - спит
  - бодрствует
  - развивается/учится новому
    9. Объект - дверь.
Состояние :
  - открыта
  - закрыта
  - цвет
  - размер
  - материал изготовления
  - дизайн
  - эксплуатация(используется по назначению/лежит на складе)
Поведение :
  - открывается(если открыть)
  - закрывается(если закрыть/запереть на ключ)
  - ломается(со временем/при неправильном использовании)
  - теряет хороший вид(старение/порча объекта)
    10. Обьект - лампочка.
Состояние : 
  - размер
  - количество ватт
  - тип(энергосберегательная/обычная/фитолампочка)
  - вкл/выкл
  - эксплуатация(вставлена в патрон люстры, работает по назначению/лежит в полке, ждет своего часа)
  - рабочая/нерабочая
  - целая/разбитая
Поведение : 
  - светит(в включенном состоянии)
  - не светит(в выключенном состоянии)
  - взрывается(от перенапряжения)
  - перегарает(закончен срок эксплуатации/неправильная эксплуатация)
  - отправляется в мусорку/переработку(в конце эксплуатации/просто надоела)
  - используется вторично(как декоративный элемент в дизайне)
