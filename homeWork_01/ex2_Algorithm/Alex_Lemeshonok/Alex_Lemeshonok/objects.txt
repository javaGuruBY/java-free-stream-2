1. Написать смс(в вк)
- Взять телефон
- Разблоктровать (разблокировался - ок, нет - конец)
- Открыть приложение ВК 
- Зайти в "личные сообщения"
- Выбрать адрессат "Иванов Иван Иванович" г.Минск (если его нет,то конец)
- Зайти в "поиск людей"
- Найти страницу "Иванов Иван Ивванович" г. Минск (если страница не найдена, то конец)
- Нажать кномку "Отправить сообщение"
- Написать любое сообщение
- Отправить сообщение (ожидаемый результат=сообщение отправлено)
2. Сделать бутерброд (батон+колбоса+огурец)
- Открыть холодильник
- Найти хлеб, огурец, колбосу (если чего-то нет то конец, утро без вкусного бутера)
- Отрезать батон (один раз)
- Отрезать колбасу (один раз)
- Отрезать огурец (3 раза)
- Положить колбасу на батон
- Положить огурец на колбасу (ожидаемый результат= готовый бутерброд батон,колбаса,огурец)
3. Создать резюме
- Открыть ноут
- Подключиться к сети
- Открыть браузер и зайти на сайт jobs.tut.by
- Нажать на кнопку "Создать резюме"
- Внести личные данные 
- Нажать на кнопку "Опубликовать резюме" (ожидаемый результат=резюме создано)
4. Купить машину 
- Приехать на рынок
- Выбрать подходящую модель 
- Проверить автомобиль
- Провести тест-драйв 
- Переоформить автомобиль
- Оплатить покупку (ожидаемый результат:машина куплена)
5. Достать деньги с капилки (Фарворовый Зайчик)
- Найти копилку (капилка есть - ок, нет -конец)
- Осмотреть капилку на наличие отверстия (если отверстие есть - достать деньги и конец)
- Разбить капилку 
- Собрать деньги 
- Убрать разбитого "зайчика" (ожидаемый результат = получены деньги с копилки)
